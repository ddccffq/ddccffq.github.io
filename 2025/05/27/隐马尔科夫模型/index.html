隐马尔科夫模型

隐马尔科夫模型的定义

隐马尔科夫模型有如下两个假设：任意状态只依赖于前一时刻状态和任意时刻的观测只依赖于该时刻的马尔科夫链的状态。
设 QQQ 是所有可能的状态的集合，VVV是所有可能的观测的集合：

Q={q1,q2,…,qN},V={v1,v2,…,vM}Q =\{q_1, q_2, \dots, q_N\}, \quad V = \{v_1, v_2, \dots, v_M\} Q={q1​,q2​,…,qN​},V={v1​,v2​,…,vM​}

其中，NNN 是可能的状态数，MMM 是可能的观测数。
III 是长度为 TTT 的状态序列，OOO 是对应的观测序列：

I=(i1,i2,…,iT),O=(o1,o2,…,oT)I = (i_1, i_2, \dots, i_T), \quad O =(o_1, o_2, \dots, o_T) I=(i1​,i2​,…,iT​),O=(o1​,o2​,…,oT​)

AAA 是状态转移概率矩阵：

A=[aij]N×N(1)A = [a_{ij}]_{N \times N} \tag{1} A=[aij​]N×N​(1)

其中，

aij=P(it+1=qj∣it=qi),i=1,2,…,N;j=1,2,…,N(2)a_{ij} = P(i_{t + 1} = q_j | i_t = q_i), \quad i = 1, 2, \dots, N; \quad j = 1, 2, \dots, N \tag{2} aij​=P(it+1​=qj​∣it​=qi​),i=1,2,…,N;j=1,2,…,N(2)

BBB 是观测概率矩阵：

B=[bj(k)]N×N(3)B = [b_j(k)]_{N \times N} \tag{3} B=[bj​(k)]N×N​(3)

其中，

bj(k)=P(ot=vk∣it=qj),k=1,2,…,M;j=1,2,…,N(4)b_j(k) = P(o_t = v_k | i_t = q_j), \quad k = 1, 2, \dots, M; \quad j = 1, 2, \dots, N \tag{4} bj​(k)=P(ot​=vk​∣it​=qj​),k=1,2,…,M;j=1,2,…,N(4)

π\piπ 是初始状态概率向量：

π=(πi)(5)\pi = (\pi_i) \tag{5} π=(πi​)(5)

其中，

πi=P(i1=qi),i=1,2,…,N(6)\pi_i = P(i_1 = q_i), \quad i = 1, 2, \dots, N \tag{6} πi​=P(i1​=qi​),i=1,2,…,N(6)

综上，可以将隐马尔科夫模型用三元符号表示，即

λ=(A,B,π)(7)\lambda = (A, B, \pi) \tag{7} λ=(A,B,π)(7)

学习算法

前文的内容是都和概率计算有关，那如何根据训练集训练一个隐马尔科夫模型模型呢？

监督学习方法

如果已给训练数据集包含 SSS 个长度相同的观测序列和对应的状态序列，我们可以用极大似然估计法来估计参数。分别的有：

\begin{align*} \hat{a_{ij}} &= \frac{A_{ij}}{\sum_{j = 1}^{N} A_{ij}}, \quad i = 1, 2, \dots, N; \quad j = 1, 2, \dots, N \tag{22} \\[10pt] \hat{b_{j}}(k) &= \frac{B_{jk}}{\sum_{k = 1}^{M} B_{jk}}, \quad j = 1, 2, \dots, N; \quad k = 1, 2, \dots, M \tag{23} \\[10pt] \hat{\pi_{i}} &= \frac{\sum \text{初始状态为$q_i$} }{S} \tag{24} \end{align*}

其中 AijA_{ij}Aij​ 表示在训练数据中，从状态 qiq_iqi​ 转移到状态 qjq_jqj​ 的次数总和。BjkB_{jk}Bjk​ 表示在状态 qjq_jqj​ 下，观测到符号 vkv_kvk​ 的次数总和。

Baum-Welch 算法

假定训练数据只包含 SSS 个长度为 TTT 的观测序列 {O1,O2,…,OS}\{O_1, O_2, \dots, O_S\}{O1​,O2​,…,OS​}，我们将观测序列数据看作观测数据 OOO，状态序列数据看做不可观测的隐数据 III，那么存在如下概率模型

P(O∣λ)=∑IP(O∣I,λ)P(I∣λ)(25)P(O | \lambda) = \sum_{I} P(O |I, \lambda) P(I |\lambda) \tag{25} P(O∣λ)=I∑​P(O∣I,λ)P(I∣λ)(25)
